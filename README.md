# Titanic Survival Prediction

## 📌 專案介紹
本專案基於 Kaggle Titanic 資料集，目標為預測乘客在鐵達尼號沉船事故中的生存機率。透過資料清理與特徵工程，搭配機器學習模型進行預測分析。

## 🔧 使用技術
- Python (Pandas)
- 機器學習：Scikit-learn (Random Forest, Logistic Regression)
